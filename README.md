КАЛКУЛАТОР - Курсова работа по Python

Описание: Проектът реализира прост калкулатор с 4 основни операции: събиране, изваждане, умножение и деление. Потребителят въвежда две числа и желаното действие, след което програмата показва резултата. Калкулаторът работи в цикъл докато не напишеш "изход".

Как се стартира?: Отваря се папката на проекта в терминал и се пише: python main.py

Библиотеки: Проектът не използва допълнителни библиотеки.
